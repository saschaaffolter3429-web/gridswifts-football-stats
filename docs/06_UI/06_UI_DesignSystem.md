# UI Design System

## Brand Colors

- Black: `#050505`
- Card: `#141416`
- Card 2: `#202024`
- Orange: `#ff7a18`
- Orange Soft: `#ffb067`
- White: `#ffffff`
- Border: `#303036`
- Field Green: `#137a36`

## UI Principles

- Fast during games
- Keyboard-first
- Minimal visual noise
- Clear hierarchy
- No giant forms
- Dynamic play-specific input panels

## Components

- Button
- IconButton
- Card
- StatCard
- Input
- Select
- PlayerSelect
- TeamBadge
- GameHeader
- PlayTimelineItem
- FieldPositionView
- WarningBanner
- Modal
- Tabs

## Keyboard Goals

- TAB moves between fields.
- ENTER confirms field.
- SPACE opens play type selector.
- ESC closes modal.
- CTRL+Z undo.
- CTRL+Y redo.
- CTRL+S save.
