# Broadcast Vision

## Goal

GridSwifts Broadcast should provide TV-quality live graphics for amateur and semi-pro football.

## Features

- Live Scoreboard
- Down & Distance
- Ballposition
- Football Field
- Animated play arrows
- Drive Summary
- Scoring animations
- Player cards
- Sponsor placements
- OBS compatible output

## Future Output Modes

- Desktop window
- Browser source for OBS
- Web ticker
- API endpoint
- HTML export
